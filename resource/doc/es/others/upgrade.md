# Método de actualización

`composer require workerman/webman-framework ^1.4.3 && composer require webman/console ^1.0.27 && php webman install`

> **Nota**
> Debido a que el proxy de composer de Aliyun ha dejado de sincronizar datos desde la fuente oficial de composer, actualmente no es posible actualizar webman a la última versión utilizando el proxy de Aliyun. Por favor utiliza el siguiente comando `composer config -g --unset repos.packagist` para restaurar el uso de la fuente de datos oficial de composer.
