# webman என்றால் என்ன

webman ஒரு உயர் செயல்பாடு HTTP சேவை உருவாக்கை ஆசிரியரான [workerman](https://www.workerman.net) அடிப்படையில் உள்ளது. webman பாம்-fpm ஆலோசனையை விரிவுபடுத்துகிறது, அதனால் அது மிகவும் உயர் செயல்படுத்தும் HTTP சேவைகளை வழங்குகிறது. நீங்கள் webman உபயோகிக்கலாம், வலைத்தள உருத்துகளை உருவாக்க, HTTP இடைப்பாடுகளையும் அல்லது சிறு சேவைகளை உருவாக்கலாம்.

அதேதரப்பின், webman தனியும் செயலாக்கப்படும் முறையில் ஆப் செயல்படுத்தும் உள்ளது, உங்கள் ஒவ்வொருவரும் சேவையையும் நுழையலாம், உதாரணமாக WebSocket சேவை, கணிவை இணைக்கோள், விளையாட்டு, TCP சேவை, UDP சேவை, ஐக்கிய சாக்கெட் சேவை போன்றவை.

# webman கொள்கை
**அதிகம் குறிக்கப்படாத ஆக்கிமாக அதிக பெருமையுடன் முறை வளர்ச்சியை ஒருமிதமாக வழங்கும்.**

வெளியென துவக்கத்தை மறுமையாக வழங்குவதாக புமிர்வுபடுவதைக் குறிக்கும். அவசியமான முறைகளை மட்டுமே வழங்குகிறது (வழு, மத்தியம், அமைப்பு, தன்னிரண்டு முறை). மற்றவை அனைத்துப் பொருள்களையும் காம்போச்ர் உருவியலில் மீளமைக்கின்றது என்றும் இதையும் குறிக்கும், உங்கள் அறிகுறிகளில் நீங்கள் webman-ல் அறிந்துகொள்வது எவ்வளை வேண்டும் என்று குறிக்கும்.

# webman-ஐ பிரயோசிக்கும் பரிமாணங்கள்

1. உயர் நிலை நிலைனாவரி. webman அடிப்படையில் உள்ள workerman மூலம் உருவாக்கப்பட்டுவருவது, workerman ஒரு குறிச்சி எதுவுமில்லாத மிகவும் உயர் நிலைப்பட்ட சாக்கெட் அமைப்புகள் உள்காட்சி.

2. அதிக உயர் செயல்பாடு. webman பாம்-fpm அமைப்புகளை 10-100 மட்டக்கண் அதிகம் செயல்படுத்துகிறது, கோகோ போர்க்ஸ் echo போன்ற வலைப்பேச்சுகளையும் அதிகமாக செயல்படுத்துகிறது.

3. அதிக மருந்தும். மாற்றில் எவ்வாறு செய்வது பெருமக்கான கோட்டவங்கள் மற்றும் வரைபடங்களையும் பரப்பவில்லை.

4. குறுக்கப்பட்ட பெருமையும் முறையும். வழங்கும் படி உங்களை  விதிவாயாக அமைப்பதில் உள்ளீர்கள், சிக்கல் எழுதல் மற்றும் பாரம் பாரதாக அந்தகூலம் இல்லாமல் உள்ளது.

5. அதிக உத்வ எளிய முறையாக, கற்றுக்கொண்டு பிற்களம் எழுத விலக்குகிறது.

6. அதிக வையகத் தெளிவில் எழுதவும்.
